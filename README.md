### React Auth System With JWT

[For more information about JWT](jwt.io)

## Techs  :rocket:
  - NodeJs
  - Express
  - JsonWebToken
  - MySQL
  - React
  - React Hooks
  
## Run  ▶️

Firstly install dependencies:

```bash
yarn install
```
and
```bash
cd client && yarn install
```

Run project (root folder):
```bash
yarn start
```
